NoobChain

NoobChain is a basic blockchain prototype built in Java to demonstrate how fundamental concepts like distributed ledgers, mining, and cryptographic validation work behind the scenes. The project features a working wallet system, transaction validation using Elliptic Curve Cryptography (ECC), and a proof-of-work consensus mechanism.
